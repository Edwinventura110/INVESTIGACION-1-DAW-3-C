INVESTIGACION - ¿QUE ES LINTING?


Primeramente ESLint es una herramienta de código abierto enfocada en el proceso de linting para javascript, esta es una herramienta predominante para la tarea de limpiar código javascript tanto en el servidor (node.js) como en el navegador. En otras palabras, es una herramienta de software que observa y revisa tu código en busca de errores que puedan afectar tu código, y existen algunos que incluso pueden darte sugerencias de como arreglar el error o incluso arreglarlo ellos mismos.

Dado que javascript es un lenguaje dinámico y de tipado débil, es especialmente fácil caer en errores humanos a la hora de escribir código. ESLint utiliza un sistema de reglas que permiten definir que es y que no es posible dentro del código.

El uso más simple que puedes darle a ESLint es revisar tu código de forma manual ejecutando un script en la terminal, para esto puedes escribir npx eslint --ext .js cada vez o configurar este comando como un script.
